Welcome to WebSmith! WebSmith is a broswer level IDE that run right within your browser. Unlike other alternatives, WebSmith provides a debugger that is directly integrated into the browser that allows you to easily resolve issues within your code. 

**Feature**
- Debugger
- Console
- Output for fast editing
- Autocorrect
- Autofill
- Easy export with individual downloads
- Runs everywhere also in mobile!
